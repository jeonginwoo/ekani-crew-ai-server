## 1. 커뮤니티 (상담 공유 게시판)

### 기능 설명
사용자가 자신의 상담 내용을 커뮤니티에 공유할 수 있다. 다른 유료 사용자들은 공유된 상담을 열람하며 비슷한 고민 사례를 참고할 수 있다.

### 공유 플로우
```
1. 상담 완료
2. "이 상담 공유하기" 버튼 클릭
3. 익명으로 커뮤니티에 게시됨
4. 다른 유료 사용자들이 열람 가능
```

### 게시글 형식
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[INFP] [연애] ESTJ 남친이랑 권태기

💬 상담 내용
ㄴ 사용자: 남친이 요즘 연락도 잘 안하고...
ㄴ AI: 그 사람이랑 어떤 관계야? MBTI 알아?
ㄴ 사용자: 3년 사귄 남친이야. ESTJ...
ㄴ AI: ...
(전체 대화 펼쳐보기)

📊 분석 결과
ㄴ 상황 정리: ...
ㄴ MBTI 궁합: INFP-ESTJ는...
ㄴ 솔루션: ...
ㄴ 대화 스크립트: ...

#권태기 #장기연애 #INFP #ESTJ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❤️ 127  👁️ 892
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 필터/검색
- **내 MBTI**: INFP가 올린 글만 보기
- **상대 MBTI**: ESTJ 관련 글만 보기
- **관계 유형**: 연애 / 우정 / 가족 / 직장
- **상황**: 권태기 / 갈등 / 고백 / 이별 등
- **인기순 / 최신순** 정렬

## 2. 프리미엄 구독

### 무료 vs 프리미엄

| 기능 | 무료 | 프리미엄 (월 4,900원) |
|------|------|----------------------|
| 상담 턴 수 | 5턴 | 50턴 |
| 분석 결과 | 기본 | 심층 (궁합 + 스크립트) |
| 내 상담 공유 | ✅ | ✅ |
| 남의 상담 열람 | ❌ | ✅ |
| 광고 | 있음 | 없음 |

### 결제 유도 포인트
1. **5턴 끝났을 때**: "더 대화하고 싶다면 프리미엄!"
2. **커뮤니티 글 클릭할 때**: "프리미엄 구독하면 다른 사람 상담도 볼 수 있어"
3. **분석 결과에서**: "궁합 분석과 대화 스크립트는 프리미엄 전용"
